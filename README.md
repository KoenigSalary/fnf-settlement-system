# 🏢 Enhanced F&F Settlement System

A comprehensive Full & Final Settlement management system built with Streamlit for Koenig Solutions.

## ✨ Features

- **Multi-month salary processing** with prorated calculations
- **Dual tax regime support** (Old & New Tax Regimes)
- **Investment deductions** for tax optimization
- **EPF calculations** with Employee Master policy integration
- **Auto-gratuity calculation** using standard formula
- **Role-based dashboards** (Payroll Team & Tax Team)
- **Professional UI/UX** with enhanced analytics
- **Real-time data persistence** between sessions
